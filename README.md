## 🧠 How the Game Works

1. Player selects Rock, Paper, or Scissors.
2. Computer randomly generates a choice.
3. The winner is determined using game rules:

   * Rock beats Scissors
   * Scissors beats Paper
   * Paper beats Rock
4. Score updates automatically.
